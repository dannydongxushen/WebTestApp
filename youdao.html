设置本地服务器指南

要解决CORS问题，最简单的方法是使用本地服务器运行此页面：

  1. 安装Node.js（从官网下载）
  2. 打开命令行/终端，导航到保存此HTML文件的目录
  3. 运行命令：npx http-server
  4. 在浏览器中访问显示的URL（通常是http://localhost:8080）

或者，您也可以使用Python：

  • Python 3: python -m http.server 8000
  • Python 2: python -m SimpleHTTPServer 8000
